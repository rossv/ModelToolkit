Purpose

The Model Toolkit is a curated directory to help users quickly discover H&H tools by purpose and platform.

  • Designed for novice to advanced modelers.
  • Current access is open. Planned enterprise authentication: Okta and Azure AD.
  • Includes tools created by the Model Toolkit maintainer and trusted third-party resources.
